"HTML 编码" 的意思是特殊的 HTML 字符替换为相应的 HTML 实体︰

* `&` 替换为 `&amp;`
* `<` 替换为 `&lt;`
* `>` 替换为 `&gt;`
* `"` 替换为 `&quot;`
