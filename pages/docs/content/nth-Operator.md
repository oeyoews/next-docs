title: nth Operator
type: text/vnd.tiddlywiki

<<.place N>> 是基于 1。换句话说， `nth[1]` 具有与 <<.olink first>> 操作符相同的效果。

<<.tip "对基于 0 的参数的等效操作符，请参阅 <<.olink zth>> 操作符">>
